## Compiling Sass

1. Install NodeJS
2. Install Node-Sass and Edje globally with `npm install -g node-sass@3.4.2` and `npm install -g edje`.
3. Restart your PC.
4. Open a command prompt in your project directory and type `npm run sass` (PC) or `npm run sass-mac` (Mac).
